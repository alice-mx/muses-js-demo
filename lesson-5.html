Task 1: Finding Elements on the page using JavaScript

You can find an element on the page using JavaScript, using code that looks like this: document.querySelector("#my-element-name"). This allows you to edit your HTML page with JavaScript.

Update document.querySelector("#I-CANT-FIND-MY-ELEMENT") to instead look for an element with the id of #update-text. When you refresh the page, the new JavaScript text should overwrite the original HTML message.
