Muses JS - HTML, CSS and JavaScript Introduction

This workshop is divided into a number of lessons, intended to teach beginners to build a simple web page.

After each lesson, you should feel confident with the 'key takeaway' dot points. If you already feel you understand these concepts, skip ahead to a lesson that is appropriate for your skill level.

Lesson 1: Basic HTML

Key Takeaways:

  • I have a text editor I'm comfortable using
  • I'm able to edit the html files in my text editor and view the changes in the browser.
  • I can create simple elements like a paragraph and heading.

Lesson 2: HTML and CSS

Key Takeaways:

  • When identifying elements, I know when to use a class and when to use an id.
  • I can apply CSS styles to HTML elements based on their classes and ids.
  • I can edit some CSS styles like background-color and width.
  • I know what comments look like in CSS.
  • I know where I can find more information about CSS properties.

Lesson 3: Interactive CSS

Key Takeaways:

  • I can apply different styles to elements when I hover over them.
  • I can add new classes to existing elements.
  • I can edit some features of an animation, like it's duration.

Lesson 4: Basic JavaScript

Key Takeaways:

  • I know where to find the JavaScript code within the module pages
  • I can edit an pop up message
  • I know what comments look like in JavaScript
  • I can find the console log in the developer tab.

Lesson 5: Editing HTML with JavaScript

Key Takeaways:

  • I can edit the document.querySelector in JavaScript to successfully find an element.
  • I can insert custom text into HTML using JavaScript.

Lesson 6: Triggering Changes with JavaScript

Key Takeaways:

  • I understand what a function is.
  • I can edit a function and make it do different things.
  • I can make a function run when a button is pressed.

Lesson 7: Variables

Key Takeaways:

  • I understand what a variable is.
  • I can use a variable to store different things, like numbers or colours.
  • I know what an array is, and can alter code to add more elements to an array.
  • I know how to access an item in an array using an index value.

Lesson 8: Loops

Key Takeaways:

  • I understand what a loop is.
  • I can use a loop to print different numbers.
  • I can use a loop to access items in an array using the index of the loop.

Lesson 9: If Statments

Key Takeaways:

  • I understand what an if statement is.
  • I can use a variable to change the result of an if statement.
  • I know how to compare numbers in a if statement, e.g.: is 'x' less than 'y'

Lesson 10: Piloting a Spaceship

  • I can connect functions to buttons.
  • I can create new functions.
  • I can use JavaScript to manipulate HTML.

Lesson 11: Web Development

Key Takeaways:

  • I can connect functions to keystrokes (e.g.: up, down, left, right).
  • I use JavaScript to manipulate HTML.
  • I can create more complicated if statements which depend on different variables.
