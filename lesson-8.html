Lesson 8 - Loops

A loop allows you to repeat an action a certain amount of times.

Task 1: Loops

Update the loop in the "runTheLoop" function to repeat 10 times, instead of once

Task 2: Changing content within a loop

Update the loop so that it prints how many times it's run, e.g.: "the loop has run 0 times, the loop has run 1 times, the loop has run 2 times"

Task 3: Looping through an array of elements

You can loop through an array of elements.
For this task, update the dynamicLoop() function so that each item in the to do list is printed. Currently, it will just print the first item out over and over again.

Key Takeaways

  • I understand what a loop is.
  • I can use a loop to print different numbers.
  • I can use a loop to access items in an array using the index of the loop.
