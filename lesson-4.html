Lesson 4 - JavaScript

What is JavaScript?

JavaScript is a language that can be used to dynamically change HTML elements. This makes webpages more interactive and useful.

Page Structure

In these modules, we've put the JavaScript after the HTML content in <script></script> tags. Sometimes JavaScript will be in another file that's imported when the page loads.

The Command Line

When programming in JavaScript, it is useful to view output in the 'console' tab of the developer window (Fn+F12 to view)

Comments

A comment is a part of the code that doesn't do anything. They start with // or /* characters. They don't change what the code does, but can be useful for putting human readable notes in the code.
