Lesson 1 - Basic HTML

This is a page

This is a simple HTML page. HTML is a 'markup language' that you can use to create the building blocks of a webpage. You can view the page in a web browser, like Chrome or Edge. To edit the page, open this file with a text editor like Notepad. It might be easier to use a text editor which is specifically for coding, like Sublime, as this will highlight parts of the code and make it easier to read.

HTML pages are made up of elements, such as h1 (heading 1), "p" (paragraph) and "div" (division).

Element can be within other elements
  • Like this list.
  • Of items.

You need additional languages like CSS and JavaScript to add style and interactivity to web pages.

Task #1

  • Add a new heading (h2) that says: HELLO WORLD!
  • Save your changes in Notepad (or whatever text editor you're using) and refresh the page in your browser.
  • You should see your new heading! Congrats!

Key Takeaways

  • I have a text editor I'm comfortable using
  • I'm able to edit the html files in my text editor and view the changes in the browser.
  • I can create simple elements like a paragraph and heading.

If you're not comfortable with any of key takeaway points, please ask your mentor for help.
