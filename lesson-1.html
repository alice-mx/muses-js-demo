Lesson 1 - HTML and CSS

You can add styles to HTML elements (like 'div' or 'h3') using CSS, which change how the elements are displayed on the page. Elements can be given ids and classes, which allow you to select them to apply styles.

Task 1: Applying styles with a class

Update the ".dotted-div" style to have a blue background and a green border.

You can select a class in CSS by adding a dot before the class name, e.g. ".my-class-name".

Task 2: Applying specific styles with an ID

Update the "task-2" style a style that makes the background of this element green, but leaves the other elements unchanged.

You can apply styles to a specific element if it has an 'id' attribute. You can select this in CSS by adding a '#' before the ID name, e.g.: '#my-element-name'.

Task 3: Nested CSS Selectors

Update the colour of this paragraph to be red. Change all other paragraphs to be dark green.

You can apply styles to a specific elements WITHIN another element. To apply styles to a paragraph within an element with an id of "my-id", you would use the following selector: "#my-id p".
