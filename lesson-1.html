Lesson 1 - HTML and CSS

You can add styles to HTML elements (like 'div' or 'h3') using CSS, which change how the elements are displayed on the page. Elements can be given ids and classes, which allow you to select them to apply styles.

Task 2: Applying specific styles with an ID

You can apply styles to a specific element if it has an 'id' attribute. You can select this in CSS by adding a '#' before the ID name, e.g.: '#my-element-name'. You should use a class when you're applying a style to many different elments. An element's id should be unique.

Task 3: Nested CSS Selectors

You can apply styles to a specific elements and classes WITHIN another element. To apply styles to an 'info' class within an element with an id of "my-id", you would use the following selector: "#my-id .info".
