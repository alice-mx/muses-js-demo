Lesson 1 - HTML and CSS

This is a simple HTML page. You can view the page in a web browser, like Chrome or Edge. To edit the page, open this file with a text editor like Notepad.

HTML pages are made up of elements, such as h1 (heading 1), "p" (paragraph) and "div" (division). You can add styles to these elements using CSS, which change how the elements are displayed on the page. Elements can be given ids and classes, which allow you to select them to apply styles.

Task 1: Applying styles with a class

Update the ".dotted-div" style to have a blue background and a green border.

You can select a class in CSS by adding a dot before the class name, e.g. ".my-class-name".

Task 2: Applying specific styles with an ID

Update the "task-2" style a style that makes the background of this element green, but leaves the other elements unchanged.

You can apply styles to a specific element if it has an 'id' attribute. You can select this in CSS by adding a '#' before the ID name, e.g.: '#my-element-name'.

Task 3: Nested CSS Selectors

Update the colour of this paragraph to be red. Change all other paragraphs to be dark green.

You can apply styles to a specific elements WITHIN another element. To apply styles to a paragraph within an element with an id of "my-id", you would use the following selector: "#my-id p".

Task 4: Applying 'hover' styles

Update this div ("#task-4") so that it has a width of 100% when the mouse hovers over it.

You can apply styles to elements which only apply at certain times. For example, if you add ":hover" at the end of a selector, those styles will only apply when you hover over an element.

Task 5: Applying simple animation

Update this div so it has an additional class, "animated-div". This will apply an existing animation to this element

Edit the animation so it repeats every 10 seconds, rather than every 2 seconds.

You can apply multiple classes to one element. Seperate the list of class names with spaces, e.g.: "class-name-1 class-2"
