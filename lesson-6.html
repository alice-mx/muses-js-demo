Lesson 6 - Triggering Changes in Javascript

It's not very interesting to have JavaScript make changes when the page loads. Luckily, we can tie these changes to be triggered by events on the page, like a button being clicked by the user.

Task 1: Functions

A function is a small bit of code that repeats the same action when called.

Update the helloWorld() function so that the pop up message says "hello world", rather than "Update this message!"

Task 2: Updating content on the page

You can use code like this document.querySelector("#my-element").innerHTML = 'example' to update content on a webpage.

Update the function to print "Hello World" to the 'update-text' paragraph

Add a function to '#clear-text-button' that removes the message from 'update-text'

Task 3: Connecting functions to buttons

Functions won't run unless they are called, or triggered by an event like pressing a button. You can use JavaScript to connect buttons to functions.

Update the button in task 4 (document.querySelector("#WHERE-IS-THE-BUTTON")) so that it's connected to #button-3 below.

Key Takeaways

  • I understand what a function is.
  • I can edit a function and make it do different things.
  • I can make a function run when a button is pressed.
